# 周线+日线策略参数优化报告

## 可调参数列表

| 参数 | 默认值 | 含义 | 优化方向 |
|------|--------|------|----------|
| `weekly_min_strokes` | 3 | 周线最小笔数 | 增加减少信号，降低噪音 |
| `daily_min_strokes` | 3 | 日线最小笔数 | 增加减少信号，降低噪音 |
| `stop_loss_pct` | 0.08 (8%) | 止损百分比 | 降低减少亏损，但可能过早止损 |
| `exit_ratio` | 0.5 (50%) | 顶背离减仓比例 | 调整分批止盈比例 |

## 参数敏感性分析

### 1. weekly_min_strokes (周线最小笔数)

| 值 | 预期效果 | 对夏普比率影响 |
|----|----------|----------------|
| 2 | 信号增多，噪音增加 | 可能降低夏普(波动增大) |
| 3 | 平衡 | 基准 |
| 4-5 | 信号减少，质量提高 | 可能提高夏普(稳定) |
| 6+ | 信号稀少 | 错失机会 |

**建议**: 趋势明显的股票用4-5，震荡市用3

### 2. daily_min_strokes (日线最小笔数)

**影响**: 影响日线卖点判断精度
**建议**: 保持与周线参数一致或略小(如周线4用日线3)

### 3. stop_loss_pct (止损百分比)

| 值 | 预期效果 |
|----|----------|
| 0.05 (5%) | 频繁止损，胜率降低，单笔亏损小 |
| 0.08 (8%) | 平衡 |
| 0.10-0.12 | 宽松止损，胜率提高，单笔亏损大 |
| 0.15+ | 止损形同虚设 |

**建议**: 波动大的股票用10-12%，波动小的用5-8%

### 4. exit_ratio (顶背离减仓比例)

| 值 | 策略含义 |
|----|----------|
| 0.3 | 保守，只减30% |
| 0.5 | 平衡，减半 |
| 0.7 | 激进，减70% |

**建议**: 高波动股票用0.7，稳健股票用0.3-0.5

## 推荐参数组合

### 组合A: 稳健型 (低波动股票)
```python
weekly_min_strokes = 4
daily_min_strokes = 3
stop_loss_pct = 0.06
exit_ratio = 0.4
```
特点: 信号质量高，止损严格，适合大盘蓝筹

### 组合B: 平衡型 (默认)
```python
weekly_min_strokes = 3
daily_min_strokes = 3
stop_loss_pct = 0.08
exit_ratio = 0.5
```
特点: 平衡收益与风险

### 组合C: 激进型 (高波动成长股)
```python
weekly_min_strokes = 5
daily_min_strokes = 4
stop_loss_pct = 0.10
exit_ratio = 0.6
```
特点: 宽松止损捕捉大趋势

## 优化目标说明

### 夏普比率公式
```
夏普比率 = (年化收益率 - 无风险利率) / 年化波动率
```

### 优化建议
1. **夏普比率 > 1.5**: 优秀策略
2. **夏普比率 1.0-1.5**: 良好策略
3. **夏普比率 0.5-1.0**: 一般策略
4. **夏普比率 < 0.5**: 需要改进

### 同时关注
- 最大回撤 (< 20% 为佳)
- 胜率 (> 40% 可接受)
- 盈亏比 (> 1.5 为佳)
- 交易次数 (> 10笔才有效)

## 注意事项

1. **过拟合风险**: 优化结果可能在样本外失效
2. **建议使用 Walk-Forward 验证**: 将数据分为样本内/样本外
3. **参数稳定性**: 好的参数对数据变化不敏感
