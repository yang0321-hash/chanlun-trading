### V11c 改动（参数扫描最优组合，3个参数）

```python
# V10 → V11c:
self.trailing_start = 0.02   # 更早启动保本(3%→2%)
self.trailing_tight = 0.025  # 保本区更紧(3%→2.5%)
self.trailing_medium = 0.06  # 正常区更宽(5%→6%)
```

**核心发现**: 参数扫描 14 个组合发现最优配置是"早启动+紧保本+宽正常区"。
- 更早启动(trailing_start 3%→2%)意味着更早锁定利润
- 更紧的保本区(tight 3%→2.5%)快速截断亏损交易
- 更宽的正常区(medium 5%→6%)给中期趋势更多呼吸空间

## 已知局限

### 日线回测无法处理盘中止损

002371 在 2025-07-04 的 -19.42% 亏损（entry=415.90, close=335.15）：

- **根因**: 回测引擎(pip包)使用收盘价成交，不支持盘中止损单
- **signal_engine fix**: 已在内部用 `max(stop_price, low)` 截断，输出 0（平仓信号）
- **但回测引擎**: 看到 signals=0 后按当日收盘价 335.15 成交，仍记录 -19.42%
- **解决**: 需要(1) 30分钟级别回测 或 (2) 改回测引擎支持止损价字段

### 持仓时间 vs 收益

| 持仓天数 | 交易数 | 平均收益 | 胜率 |
|---------|--------|---------|------|
| 0~5天 | 21 | +2.85% | 81% |
| 5~10天 | 23 | +2.54% | 78% |
| 10~20天 | 55 | +1.45% | 65% |
| 20~40天 | 75 | +6.72% | 81% |
| 40~60天 | 23 | +10.97% | 100% |
| >60天 | 17 | +16.79% | 94% |

**结论**: 中长期持仓(20天+)收益和胜率显著高于短期。短持仓多为噪声交易。

## 优化方向

1. **V11 过滤噪声入场**: 增加 MA20 趋势确认或量能门槛，减少 -3~0% 区间的 31 笔小亏
2. **V11 改善退出**: 短期(10天<)退出条件收紧，避免 10~20天区间 65% 胜率的低质量退出
3. **30分钟回测**: 解决盘中止损问题
