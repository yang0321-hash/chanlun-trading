# 缠论稳定盈利交易系统使用指南

## 一、系统核心理念

### 1.1 交易哲学

本系统基于缠论理论的核心思想，构建了一套可复制的稳定盈利交易体系：

- **多级别联立**：周线定方向，日线找买点，30分确认入场
- **三类买点优先**：2买 > 3买 > 1买（安全性递减，收益潜力递增）
- **中枢操作**：在中枢下沿买入，中枢上沿卖出
- **背驰确认**：MACD背驰确认转折
- **严格止损**：每笔交易风险不超过2%

### 1.2 缠论核心概念回顾

```
分型（顶分型/底分型）
    ↓
笔（连接顶底分型）
    ↓
线段（笔的延伸）
    ↓
中枢（至少三笔重叠区域）
    ↓
买卖点（基于中枢和背驰）
```

#### 三类买点定义

| 买点类型 | 定义 | 安全性 | 收益潜力 |
|---------|------|--------|----------|
| 1买 | 下跌趋势中，最后一个中枢下移后的背驰点 | 低 | 高 |
| 2买 | 1买后的回踩，不创新低 | 高 | 中 |
| 3买 | 突破中枢后，回踩不回中枢 | 中 | 中高 |

## 三、参数配置指南

### 3.1 风险参数

| 参数 | 默认值 | 说明 | 建议范围 |
|-----|--------|------|----------|
| max_risk_per_trade | 0.02 | 单笔最大风险 | 0.01-0.03 |
| max_drawdown_pct | 0.15 | 最大回撤后暂停 | 0.10-0.20 |
| stop_loss_atr_multiplier | 2.0 | ATR止损倍数 | 1.5-3.0 |

**配置原则**：
- 保守型：risk_per_trade=0.01, drawdown=0.10
- 平衡型：risk_per_trade=0.02, drawdown=0.15
- 激进型：risk_per_trade=0.03, drawdown=0.20

### 3.2 买入参数

| 参数 | 默认值 | 说明 |
|-----|--------|------|
| enable_buy1 | True | 是否交易1买（高风险高收益） |
| enable_buy2 | True | 是否交易2买（推荐） |
| enable_buy3 | True | 是否交易3买 |
| min_confidence | 0.6 | 最低信号置信度 |

**建议配置**：
- 新手：enable_buy1=False, enable_buy2=True, enable_buy3=True
- 稳健：全部启用，min_confidence=0.65
- 激进：全部启用，min_confidence=0.5

### 3.3 止盈参数

| 参数 | 默认值 | 说明 |
|-----|--------|------|
| trailing_stop_pct | 0.05 | 移动止损幅度（5%） |
| trailing_activate_pct | 0.10 | 盈利10%后启用移动止损 |
| partial_exit_ratio | 0.5 | 顶背离减仓比例 |

## 五、交易规则详解

### 5.1 入场条件

**多级别共振**：
1. 周线：趋势向上或形成底部中枢
2. 日线：出现2买或3买信号
3. MACD：金叉或底背离
4. 价格：在中枢下沿附近或突破前高

### 5.2 出场条件

1. **止损**：
   - 1买：跌破前一中枢下沿
   - 2买：跌破1买低点
   - 3买：跌回中枢内
   - 或亏损5-8%

2. **移动止损**：
   - 盈利>10%后启用
   - 最高价回撤5%止损

3. **止盈**：
   - 目标位：前高或中枢上沿+15%
   - MACD顶背离减仓50%
   - 日线2卖清仓

### 5.3 仓位规则

- 2买：80%仓位（高确定性）
- 3买：60%仓位（中等确定性）
- 1买：40%仓位（低确定性）
- 单笔风险控制在2%以内

## 六、注意事项

### 6.1 风险提示

1. **回测≠实盘**：历史表现不代表未来收益
2. **滑点影响**：实盘滑点可能比回测更大
3. **流动性**：大盘股比小盘股更易执行
4. **极端行情**：涨跌停、停牌等特殊情况

### 6.2 优化建议

1. **调整参数**：根据个人风险偏好调整止损和仓位
2. **股票筛选**：选择流动性好、波动适中的股票
3. **多品种组合**：分散投资，降低单一股票风险
4. **定期复盘**：每月检查交易记录，优化策略

### 6.3 常见问题

**Q：为什么没有交易信号？**
A：可能原因：
- 数据不足（需要至少60根日线）
- 置信度阈值太高
- 市场不满足买入条件（如强下降趋势）

**Q：胜率为什么不高？**
A：缠论2买、3买追求的是盈亏比，不是高胜率。即使胜率只有40%，只要盈亏比>2，长期仍然盈利。

**Q：如何提高收益？**
A：
1. 降低min_confidence（增加交易频率）
2. 启用1买（高风险高收益）
3. 调整止盈目标（更激进）
4. 优化选股（选择波动大的股票）

## 七、实盘部署建议

### 7.1 部署前准备

1. **模拟盘验证**：先模拟运行3-6个月
2. **参数调优**：根据模拟结果调整参数
3. **风控测试**：测试最大回撤是否在承受范围
4. **券商接口**：准备实盘交易接口

### 7.2 实盘运行

1. **每日收盘后**：运行选股，获取次日关注列表
2. **盘中监控**：实时监控信号，手动确认下单
3. **持仓管理**：严格按照止损止盈规则执行
4. **定期复盘**：每周回顾交易，总结经验
