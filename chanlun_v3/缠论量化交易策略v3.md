## 二、多级别联立框架

**核心原则：周线看方向 → 日线找结构 → 30分钟定入场**

### Level 1: 周线（看方向）

| 方法 | 判定 | 说明 |
|------|------|------|
| **周线方向** | GG/DD法 | 最新周线笔方向：高点抬高=UP，低点降低=DN |
| **周线位置** | 仅参考不过滤 | AB测试732个3B信号，周线位置无显著区分度 |
| **周线背驰** | 区间套辅助 | 周线背驰+日线3B = 最强组合 |

### Level 2: 日线（找结构+买卖点）

- 识别中枢（ZG/ZD动态更新）
- 检测123买/卖点（缠论原文定义）
- 分型强度加权（强/标准/弱，±0.05~0.10置信度）

### Level 3: 30分钟（定精确入场）

**缠论原文：大级别买点 = 次级别1B**

| 入场条件 | 3d胜率 | 说明 |
|----------|--------|------|
| **30分钟1B/2B买点** | 最高 | 缠论原文严格入场，Step 0最高优先级 |
| **背驰 + 强分型** | 93.0% | 直接入场 |
| **背驰 + 弱分型** | ~90% | 可入场 |
| **无背驰 + 弱分型** | 86.1% | **不入场** |

### 30分钟回调约束

30分钟回调不能跌破日线中枢ZG：

| 情况 | 判定 | 操作 |
|------|------|------|
| 回调低点 > ZG | ✅ 正常入场 | 按信号操作 |
| 回调低点 < ZG，回抽 < GG | 🔴 **2卖确认** | 清仓 |
| 回调低点 < ZG，回抽 ≥ GG | ⚠️ **2卖预警** | 减仓观察 |

## 三、出场系统（ExitSystem）

### 3.1 分批止盈 + 背驰回补（主策略）

> AB测试：胜率99.0%，均收+14.15%（vs 买入持有90.2%/+9.61%）

```
阶段1: HOLDING（满仓100%）
  │
  ├─ 跌破ZG → 清仓止损（全部亏损）
  │
  └─ 第1笔UP反弹完成
       │
       ▼
阶段2: TP50（已止盈50%，持有50%）
  │
  ├─ 跌破ZG → 清仓剩余50%
  │
  └─ 3笔回调(DN-UP-DN)完成
       │
       ├─ MACD背驰（后段面积<前段）→ 回补50% → 阶段3
       │
       └─ 无背驰 → 不回补，保持50% → 阶段4
       │
       ▼
阶段3: RELOADED（满仓100%）
  │
  └─ 跌破ZG → 清仓止损

阶段4: NO_DIV（50%仓位不再变动）
  │
  └─ 跌破ZG → 清仓止损
```

**关键参数：**
- 止损位：max(ZG × 0.99, 买入价 × 0.99)
- 止盈触发：第1笔UP反弹的最高点
- 回补条件：3笔回调 + MACD面积背驰
- 30天到期自动平仓

### 3.2 传统出场（备用）

| 信号 | 条件 | 置信度 |
|------|------|--------|
| 硬止损 | 跌幅 > 5% | 1.0 |
| 结构止损 | 跌破买点止损位 | 1.0 |
| BIAS超买 | BIAS6 > 8% | 0.5~0.9 |
| 跟踪止盈 | 盈利>5%后回撤>3% | 0.7 |
| ATR止损 | 突破ATR×1.5 | 0.65 |

## 六、止损详解

### 各买点止损逻辑

| 买点 | 止损公式 | 含义 |
|------|----------|------|
| **1B / sub1B** | `max(背驰DN笔低点×0.99, 买入价×0.95)` | 跌破背驰低点=趋势背驰失效，取结构止损和价格止损的较高者 |
| **2B** | `回调DN笔低点×0.99` | 跌破二买回调低点=二买结构破坏 |
| **2B3B** | `1买价格×1.01` | 跌破一买价格=二买三买重合结构完全失效 |
| **3B** | `max(ZG×0.99, 买入价×0.99)` | 跌破中枢ZG=三买失效，取结构止损和价格止损的较高者 |
| **xzd1B** | `买入价×0.97` | 小转大信号止损3% |
| **pz1B** | `买入价×0.97` | 盘整背驰止损3% |
| **quasi2B / subQuasi2B** | `回调低点×0.99` | 跌破类二买回调低点=失效 |

### 卖点回补止损

| 卖点 | 回补止损 | 含义 |
|------|----------|------|
| **1S / sub1S** | `卖出价×1.01` | 价格涨回卖出价以上=卖点失效 |
| **2S** | `卖出价×1.01` | 同上 |
| **3S** | `min(ZD×1.01, 卖出价×1.01)` | 价格涨回ZD以上=三卖失效 |

## 八、完整交易流程

```
每日15:30 全A股扫描
    │
    ▼
周线方向判断（GG/DD法）
    │
    ▼
日线结构分析 → 123买卖点
    │
    ▼
30分钟精确入场
    ├─ 有1B/2B → 缠论原文严格入场 ✅
    ├─ 背驰+强分型 → 备选入场 ✅
    └─ 无背驰+弱分型 → 不入场 ❌
    │
    ▼
买入后 → ExitSystem分批止盈
    ├─ 第1笔UP完成 → 止盈50%
    ├─ 3笔回调背驰 → 回补50%
    └─ 跌破止损位 → 清仓止损
    │
    ▼
持仓监控 → 卖点检测
    ├─ 1S/sub1S → 减仓50%/30%
    └─ 2S/3S → 清仓
```
