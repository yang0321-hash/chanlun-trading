# 缠论交易决策 Skill

对单只或多只股票进行多周期缠论综合分析，给出可操作的交易建议（买入/卖出/持仓/关注）。

## 使用方法

### 基础分析（日线+周线）

```
/trade sz002600
```

### 指定时间框架

```
/trade sz002600 --tf 30min,daily,weekly
```

### 仅日线分析

```
/trade 600519 --tf daily
```

### 批量扫描

```
/trade sz002600,sh600519,300936 --batch
```

## 命令参数

- `code`: 股票代码（必需），逗号分隔支持批量。格式：sz002600、sh600519、000001.SZ、600519
- `--tf`: 时间框架，逗号分隔（默认 daily,weekly），可选：30min, daily, weekly
- `--capital`: 模拟资金量（默认 100000）
- `--batch`: 批量模式，输出简要汇总表

## 输出内容

### 单股详细报告

1. **当前价格** — 最新收盘价和日期
2. **操作建议** — 买入/卖出/观望/关注 + 信号类型（1买/2买/3买等）
3. **多周期分析** — 周线战略方向 + 日线操作信号 + 30分钟入场确认
4. **缠论结构** — 分型、笔、中枢、买卖点、价格与中枢距离
5. **MACD状态** — DIF/DEA/柱状图、金叉/死叉、背驰
6. **风控建议** — 结构止损、目标价、盈亏比
7. **仓位计算** — 基于风险敞口的建议仓位
8. **趋势判断** — 无买卖点时给出关注/警惕/观望建议

### 批量扫描表

- 股票代码 | 价格 | 操作 | 趋势 | 置信度 | 原因

## HOLD 决策三级建议

当无近期买卖点信号时，系统根据趋势+MACD+中枢距离给出分级建议：

- **关注回调买点** — 趋势偏多 + MACD多头
- **关注反弹** — 趋势偏空 + MACD空头减弱
- **多头动能减弱，注意风险** — 趋势偏多但MACD减弱
- **警惕继续下跌** — 趋势偏空 + MACD空头
- **等待结构形成** — 震荡，无明确方向
