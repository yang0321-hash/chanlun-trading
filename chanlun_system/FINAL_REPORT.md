## v9 核心优化 (v8→v9)

### T1.1 分阶段移动止损
- **v8**: 盈利6%启动, 固定回撤5%平仓 → 小震洗出大趋势
- **v9**: 盈利3%启动(更早保本), 分三档回撤容忍:
  - 盈利3-6%: 回撤3%平仓(保本区, 快速锁利)
  - 盈利6-15%: 回撤5%平仓(正常区)
  - 盈利>15%: 回撤7%平仓(大趋势区, 让利润奔跑)
- **效果**: 长仓平均盈利从7.7%提升, 年化+6.2%

### T1.2 背驰信号加成 + 三重弱势过滤
- **背驰加成0.18**: 底背驰信号最低仓位0.18(比v8的0.15更高)
- **三重弱势过滤**: MACD弱 + 量能弱(<0.5倍) + MA20弱势(价格<93%MA20) + 无背驰 → 跳过
- **教训**: 信号分级(A/B/C三档仓位)过度限制交易导致收益下降, 保留v8加法模型+弱信号过滤更优

## v9 信号引擎架构

```
输入: OHLCV日线
↓
预计算: 笔检测(bi_buy/bi_sell) + MACD(DIF/DEA/Hist) + ATR + MA20 + 量MA20
↓
逐K线模拟:
  持仓中:
    1. 结构止损(ATR动态)
    2. 分阶段移动止损(3%/5%/7%三档)
    3. 顶背驰卖出(盈利>3%+顶背驰)
    4. 2卖出局(bi_sell)
    5. 时间止损(60天)
    6. 减仓50%(盈利>4%+bi_sell)
    7. 盈利加仓(盈利>5%+bi_buy)
  空仓中:
    过滤: 回撤暂停 / 持仓上限 / 冷却期 / 周线非跌 / 涨停 / 三重弱势
    仓位: 加法模型(base+macd+vol) + 背驰加成0.18
    MA20弱势: 非背驰信号降为最小仓位
```
