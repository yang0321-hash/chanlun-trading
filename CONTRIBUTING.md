# Contributing to ChanLun Trading System

感谢你考虑为 ChanLun Trading System 贡献！

## 开发环境设置

```bash
# 克隆仓库
git clone https://github.com/你的用户名/chanlun-trading.git
cd chanlun-trading

# 安装依赖
pip install -r requirements.txt

# 运行测试
pytest tests/
```

## 贡献指南

### 代码规范

- 使用有意义的变量和函数名
- 添加必要的注释和文档字符串
- 遵循 PEP 8 代码风格

### 提交规范

```
feat: 添加新功能
fix: 修复 bug
docs: 更新文档
style: 代码格式调整
refactor: 重构代码
test: 添加测试
chore: 构建/工具链更新
```

### Pull Request 流程

1. Fork 本仓库
2. 创建特性分支 (`git checkout -b feature/AmazingFeature`)
3. 提交更改 (`git commit -m 'feat: add amazing feature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 创建 Pull Request

## 报告问题

请使用 GitHub Issues 报告 bug 或提出功能请求，并提供：
- 问题描述
- 复现步骤
- 预期行为
- 实际行为
- 环境信息（Python 版本、操作系统等）
